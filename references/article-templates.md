# 記事テンプレート集

## テンプレート 1：ハウツー記事（技術的な手順を解説する）

```markdown
# [ツール/技術] で [目的] を実現する方法

## はじめに

この記事では、[ツール/技術] を使って [目的] を実現する手順を解説します。

**対象読者：**
- [前提知識 1] がわかる方
- [前提知識 2] を使ったことがある方

**この記事を読むとわかること：**
- [ポイント 1]
- [ポイント 2]
- [ポイント 3]

## 環境・前提条件

| 項目 | バージョン / 条件 |
|---|---|
| OS | macOS 14 / Ubuntu 22.04 |
| [ツール名] | v[X.Y.Z] |

## 手順

### 1. [ステップ 1 のタイトル]

[説明文]

```bash
# コマンド例
command --option value
```

### 2. [ステップ 2 のタイトル]

[説明文]

### 3. [ステップ 3 のタイトル]

[説明文]

## 動作確認

[確認手順と期待される出力を記載]

```bash
# 確認コマンド
```

## まとめ

この記事では [ツール/技術] を使って [目的] を実現する方法を解説しました。

- [達成したこと 1]
- [達成したこと 2]

次のステップとして [関連トピック] も参考にしてください。

## 参考文献

- [公式ドキュメント](URL)
- [関連記事](URL)
```

[定義と背景を 3〜5 段落で]

[モチベーション・解決する課題]

[アナロジーや図を使った説明]

## [概念名] の仕組み

[内部動作・アーキテクチャ]

## 実際に使ってみる

[ハンズオン例]

## よくある誤解

| 誤解 | 正しい理解 |
|---|---|
| [誤解 1] | [正解 1] |

## まとめ

[概念名] について [3〜5 行でまとめ]。

## Zenn 固有記法チートシート

```markdown
# メッセージボックス
:::message
通常のメモ・補足情報
:::

:::message alert
警告・注意事項（赤系）
:::

# アコーディオン（折りたたみ）
:::details タイトル（クリックで展開）
内容
:::

# 画像キャプション
![代替テキスト](URL)
*キャプション*
```
